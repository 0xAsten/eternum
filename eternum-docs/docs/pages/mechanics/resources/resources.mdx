# Resources

> Eternum features a complex resource system where each resource type serves different purposes in the game economy.
>
> Resources exist as in-game assets that can be produced and traded, and can be converted to ERC20 tokens when bridged
> out of the game through the central bank.

## Resource Categories

### Food

- Freely producible by all players

### Currency
