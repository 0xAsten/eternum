# Introduction to Eternum

## Seasons

- The game runs in seasons, each with a defined start and end
- Entry requires a Season Ticket, minted from original Loot Realms NFTs
- Players can start with resources bridged from previous seasons or purchased.
- Compete for victory through various strategic paths


## Realms

- NFTs that determine your resource production capabilities
- Each Realm has access to specific resources from a pool of 22 different types
- Your base of operations for building and development

## The World Map

- Infinite hexagonal grid
- Procedurally generated terrain
- Strategic positioning for defense and expansion

## Resources & Economy

- 22 different resource types as ERC20 tokens
- Player-driven markets
- Strategic trading essential for acquiring resources you can't produce
- Resources can be bridged out of the game during a season.

## Military Power

- Army building and combat
- Territory control
- Strategic warfare

## Social Structure

- Tribe formation
- Diplomatic relations
